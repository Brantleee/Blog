My First HTML Boxing Match

CS347

Week two of learning HTML certainly ramped up quickly, as we began controlling where images are placed within the body, as well as implementing mox models. These were both very interesting (and difficult) to learn, however for me, the star of week two was CSS. Previosly I had to painstakingly define each elements attributes within a container, however with CSS I was able to apply style in more complex ways, covering multiple aspects of the HTML file. This was especially useful in the Box Model Lab, as I used CSS to define the style for the cards, then simply applied that style guide to each card I created. CSS is now a necessity for any HTML code that has multiple containers, images, or a specific layout I must adhere to.

These are some of the resources I used to help me through week two.
How to add code in-line
I was having difficulty adding "<" and ">" into my HTML, without it being interpreted as code. Thankfully devPractical had a detailed and useful guide.
Making cards with CSS
